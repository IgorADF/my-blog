# HTML English Labels

When generating any HTML, JSX, or Astro markup, all hardcoded strings must be in English.

This includes: button labels, form labels, placeholders, headings, error/success messages, `aria-label`, `alt`, `title`, tooltips, empty states, and any other visible or accessible text.

The language of the conversation does not affect this rule.
